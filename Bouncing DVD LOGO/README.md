# Bouncing DVD Logo

A Python project that simulates the classic bouncing DVD logo animation.

## Features

- Bounces around the screen with realistic physics
- Changes color on wall collisions
- Configurable window size and speed

## Requirements

- Python 3.7+
- pygame

## Installation

```bash
pip install pygame
```

## Usage

```bash
python main.py
```

## Controls

- Close the window to exit
